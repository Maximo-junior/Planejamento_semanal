Sobre o projeto:

Usei o vite React na construção desse projeto,
É uma ferramenta para o frontend JavaScript com a qual 
você pode gerar estrutura de código de vários frameworks como React, Vanilla, Vue, Svelte e outros

Isso basicamente permite que o navegador assuma parte do trabalho de um empacotador, por exemplo, Webpack. Assim reduzindo drasticamente o tempo de compilação 
no ambiente de desenvolvimento.
